# ADR 010: Developer Tools track

- Status: Accepted
- Date: 2026-07-15

## Decision

Enter OpenAI Build Week in the Developer Tools track.

## Why

The personalized artist website proves the platform works. The Build Week invention is the transferable, agent-operable repository: configuration schemas, deterministic setup, security boundaries, media processing, service runbooks, validation, portability, and ongoing Codex maintenance.

## Consequences

The judge path must let reviewers test the working result without rebuilding it. The README and video must show both the artist-facing outcome and the repository's ability to explain, configure, verify, and maintain itself.
